# C Tic-Tac-Toe

A simple **terminal-based Tic-Tac-Toe game written in C**.

I built this project mainly as a way to get more comfortable working with the **C programming language**, focusing on fundamentals like memory handling, arrays, pointers, and general program structure.

Instead of relying on libraries or frameworks, everything is implemented using **standard C and console output**, making it a great exercise in understanding how low-level logic works.

---

## Why I Built This

While learning C, I wanted a small project that would force me to practice core concepts repeatedly. Tic-Tac-Toe is perfect for that because it requires:

* Managing a 2D board
* Handling user input
* Implementing game logic
* Detecting win conditions
* Managing turns between players

The goal wasn’t to build the most advanced version of the game, but to **get comfortable thinking in C** and writing clean procedural logic.

---

## Features

* Terminal / console based gameplay
* Two-player turn system
* Board rendering in the terminal
* Win condition detection
* Input-based move selection

## Concepts Practiced

Some of the C concepts explored in this project include:

* Pointers

* User input handling

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/RTBlanco/C_tic-tac-toe.git
cd C_tic-tac-toe
```

### 2. Compile

```bash
gcc main.c -o tic-tac-toe
```

### 3. Run

```bash
./tic-tac-toe
```
